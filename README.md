Student Study Helper
Project Overview

Student Study Helper is a web application that helps students study more efficiently by using Artificial Intelligence to analyze course notes. Users can paste their study notes into the application, generate concise summaries, extract key points, create AI-generated review questions and answers, and save their study sessions for future reference.

The application uses Microsoft Foundry (Azure AI) for text generation and Azure Blob Storage for persistent storage of study sessions.

Features
📄 Paste lecture notes or study material
🤖 Generate an AI-powered summary
🔑 Extract important key points
❓ Generate review questions with answers
💾 Save study sessions to Azure Blob Storage
📚 View previously saved study sessions in the History page
🌐 Responsive web interface built with HTML, CSS, and JavaScript
